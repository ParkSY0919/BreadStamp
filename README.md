# 빵도장 (BreadStamp)

동네 빵집 정복 스탬프 북 - iOS App

---

## 소개

빵집 방문 시 스탬프를 모으고, 먹은 빵을 수집하는 게임화된 기록 앱입니다.
지역별 빵집 정복률을 시각화하고, 빵 도감을 완성해가는 재미를 제공합니다.

## 주요 기능

- 🏪 **빵집 스탬프** - 빵집 방문 시 스탬프 획득, 지역별 정복률 표시
- 🍞 **빵 도감** - 먹은 빵 종류별 수집, 사진/평점 기록
- 🗺️ **지도 뷰** - 방문한 빵집 지도에 표시, 미방문 빵집 추천
- 🏆 **통계/업적** - 총 방문 수, 먹은 빵 종류, 업적 배지

## 기술 스택

- SwiftUI
- SwiftData
- MVVM Architecture
